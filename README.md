## 一、项目介绍

- 时间：2022.01 - 2022.07

- 概述：《治愈小镇》是一款以**园艺疗法**为理论核心的VR心理治疗游戏。玩家在宁静的虚拟小镇中体验翻土、播种、浇水、等待植物生长、收获果实等完整园艺过程，通过沉浸式的自然交互缓解焦虑与抑郁情绪，获得放松与成就感。

## 二、我的角色和贡献

- **独立完成全流程开发**（毕业设计，导师指导）

- **理论研究**：梳理园艺疗法、心流理论，提取园艺活动要素

- **游戏设计**：剧情策划、玩法机制、数值平衡、交互流程

- **美术与建模**：场景搭建、角色动画绑定（Mixamo）

- **程序实现**：所有C#脚本编写（道具系统、时间系统、种植系统、VR交互等）

- **测试与验证**：用例测试、SCL-90用户实验与数据分析

## 三、核心玩法与设计亮点

### 1.完整的园艺种植流程

- 翻土 → 播种 → 浇水 → 植物生长（种子/小苗/开花/成熟） → 采摘果实

- 每种植物有独立的生长周期（对应现实时间2~6分钟）

### 2.游戏时间与日夜循环

- 游戏时间流速可调（最终设定现实0.3秒 = 游戏内1分钟）

- 阳光方向随游戏时间旋转，营造昼夜交替的真实感

### 3.沉浸式背包与道具系统

- 通过手柄射线打开背包，悬浮显示道具详情

- 手持道具与背包内道具一键交换，工具/种子/果实分类管理

### 4.NPC引导与剧情叙事

- 新手引导场景：学习移动、旋转、传送、抓握等基础操作

- 主要剧情：通过NPC发放任务，串联园艺活动，增强代入感

### 5.心流理论融合

- 明确目标（接取任务→种植特定植物）

- 即时反馈（土壤状态变化、植物生长动画、音效提示）

- 挑战与技能平衡（等待时间适中，操作难度可调）

## 四、技术实现亮点

| 模块         | 技术方案                                     |
| ------------ | -------------------------------------------- |
| VR开发框架   | SteamVR Plugin                               |
| 编程语言     | C#（Unity引擎）                              |
| 数据存储     | ScriptableObject（道具、种子、工具数据驱动） |
| 植物生长系统 | 多阶段模型切换 + 基于时间差的状态机          |
| 土地交互     | 射线检测 + 三种土壤材质（普通/可种植/湿润）  |
| 背包UI       | 世界空间Canvas + VR插件实现射线点击/悬浮     |
| 角色动画     | Animator状态机 + Mixamo导入动画              |
| NPC寻路      | NavMesh Agent自动导航                        |
| 日夜循环     | Directional Light旋转（-90° → 90°）          |

## 五、测试与验证

### 1.运行测试

- 全用例覆盖（开始/新手/种植场景，共20+项）

- 修复卡顿、场景切换过暗、角色高度不合理等问题

### 2.功能效果测试

- **工具**：精神症状自评量表（SCL-90）

- **对象**：具有焦虑/抑郁情绪的大学生

- **结果**：
  - 焦虑因子：1.60 → 1.00
  - 抑郁因子：1.62 → 1.15
  - 人际关系敏感：1.44 → 1.22

- **用户反馈**：可玩性7/10，放松程度7/10，操作略有难度（未来优化方向）
